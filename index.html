DHCP

El protocolo DHCP (Dynamic Host Configuration Protocol) asigna automáticamente direcciones IP a los dispositivos de una red.

Ventajas

- Evita conflictos de direcciones IP.
- Usa un modelo cliente-servidor.
- Facilita la administración centralizada de direcciones.
- Permite la integración automática en la red.

Funcionamiento

- Un servidor DHCP asigna direcciones dentro de un rango prefijado.
- Si un cliente no encuentra un servidor DHCP, no podrá conectarse a la red.
- Si lo encuentra, recibirá una configuración IP temporal y luego solicitará una renovación.

Asignaciones

Asignación manual: El administrador configura la IP manualmente en el servidor.

Asignación automática: El servidor asigna una IP permanente al primer contacto del cliente.
