HTML, siglas de HyperText Markup Language, es el lenguaje estándar para estructurar el contenido de las páginas web. Define el contenido como texto, imágenes, enlaces y otros elementos usando un sistema de etiquetas que le dicen al navegador cómo organizarlos y presentarlos al usuario

CSS, o Hojas de Estilo en Cascada, es un lenguaje de estilos que define la apariencia de los documentos HTML, como el color, la fuente, el tamaño, la posición y el diseño de una página web
